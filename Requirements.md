# Requirements

## Functional Requirements

1. The system shall generate a target number between 1 and 100.
2. The player shall be given a maximum of 7 attempts to guess the number.
3. The system shall accept numerical input from the user.
4. The system shall validate that the entered number is between 1 and 100.
5. The system shall display:

   * "Too High!" when the guess is greater than the target number.
   * "Too Low!" when the guess is less than the target number.
6. The system shall display a success message when the correct number is guessed.
7. The system shall display the remaining number of attempts after each valid guess.
8. The system shall track and display the player's score.
9. The system shall allow the player to start a new game after each round.
10. The system shall handle invalid inputs using exception handling.

## Non-Functional Requirements

1. The application shall run in a Java console environment.
2. The program shall provide a user-friendly interface.
3. The system shall respond immediately to user input.
4. The application shall be lightweight and require minimal resources.
5. The code shall follow Java coding standards and best practices.

## Software Requirements

* Java JDK 8 or above
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code, NetBeans) or Command Prompt/Terminal

## Hardware Requirements

* Processor: Any modern CPU
* RAM: 2 GB or higher
* Storage: 50 MB free space
